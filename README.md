# Interception.RocketFasterPermissions
Something like a wrapper for RocketMod's permissions/groups 
Actually just an attempt to make them work faster using dictionaries and hashsets instead of lists

## Dependencies
This project uses few more external libraries to work
* [Interception.Module](https://github.com/interception-plugins/Interception.Module)
* [RocketMod itself ofc](https://github.com/SmartlyDressedGames/Legally-Distinct-Missile)

## Installation
* __Interception.RocketFasterPermissions.dll__ must be moved into both __Rocket/Libraries__ and __Rocket/Plugins__ directories
